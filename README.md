# Patri Star Bot

Telegram chatbot. Upon user request, sends "astronomy picture of the day" with a brief explanation.

The bot used open API <a href="https://github.com/nasa/apod-api">Astronomy Picture of the Day (APOD) microservice</a>.

You can see the bot in action in Telegram by username <a href="https://t.me/PatriStar_bot">@PatriStar_bot</a>.
